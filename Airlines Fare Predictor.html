d. If the goal of the analysis was to evaluate the effect of Southwest Airlines’ presence on the airline industry rather than predicting fares on new routes, how would the analysis be different? Describe technical and conceptual aspects.¶

This is an example of the difference between explanatory (descriptive) modeling and predictive modeling. First, if we care about the effect of SW presence in the industry, we would not care about the predictive power of the model, AKA finding y-bar. Instead, we would care about the coefficients of the features, AKA the betas. In doing so, we wouldn't need to split our data into training and validation subsets. We would use as much data as we can to train the model, to 'evaluate' the betas. Also, we wouldn't use the accuracy is finding the best model, we would care more about the fit measures of the data.

Second, if we care about evaluating the effect of SW, we would care much about the average values of most features. Hence, not a case-by-case measure.

Our goal here is to predict whether an accident just reported will involve an injury (MAX_SEV_IR = 1 or 2) or will not (MAX_SEV_IR = 0). For this purpose, create a dummy variable called INJURY that takes the value yes if MAX_SEV_IR = 1 or 2, and otherwise no.
